
## Initialization
1. Install the libraries form folder lib 
2. Input the config parameter { mqttServer,mqttUser,mqttPassword,device_name,topic }
3. upload the code to esp32

## step to use
1. connect the powersupply of usb type-c to esp32
2. connect the wifi of esp name "ESP32-Setup"
3. select your wifi 
4. esp32 connecting to mqtt broker 

## reconnect wifi router
1. if you want to reconnect the wifi press the sw IO "0" 3 sec 
2. the wifi manager will be reset the config and bring up the " ESP32-Setup page "
